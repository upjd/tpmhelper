#根据作业组属性统计不良率

#需求

根据作业组属性检测结果的值分析出检测总数、不良数、不良率

#需求明细

  • 可用维度：时间、组织单元、人员、作业组
  • 维度来源：默认
  • 抽取作业组：全部
  • 检测结果不良ID：b53c356c-f525-44a7-b675-fdb5d664bfd5
  • 子KPI：检测总数、不良数、不良率
  • 计算公式：不良率 = 不良数 / 检测总数
